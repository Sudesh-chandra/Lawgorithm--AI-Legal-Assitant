# ⚖️ Lawgorithm – AI-Powered Legal Documentation Assistant

Your one-stop platform for **smart legal document generation, expert guidance, and AI-powered chat support** — simplifying legal workflows for individuals and businesses.

---

## 🚩 Problem Statement

Legal documentation is often slow, expensive, and confusing. Individuals and businesses struggle with:

- Understanding legal terminology  
- Finding the right document templates  
- Accessing reliable legal guidance  
- Customizing contracts for specific roles (HR, SEO, Accounting, etc.)

**Lawgorithm** solves these challenges by combining AI, automation, and a network of legal experts to deliver accurate, personalized documentation — in minutes, not days.

---

## 💡 Solution Overview

Lawgorithm is modular, user-friendly, and legally aware. It comprises three main pillars:

### 📄 1. Document Generator

- Select your role (e.g., HR, Accountant)
- Fill out a smart, contextual form
- Instantly receive a tailored legal document (PDF/DOCX)

### 🤖 2. AI Chatbot Assistant

- Powered by GPT  
- Explains legal terms, formats, and provides drafting suggestions  
- Helps users understand and improve their documents

### 👩‍⚖️ 3. Lawyer Connect ("Corner of Experts")

- Curated network of legal advisors
- Connect directly for reviews or advanced help

---

## ✨ Features

- 🔠 **Role-Based Document Templates** (HR, SEO, Finance, etc.)
- 💬 **AI Chatbot** for legal Q&A and drafting support
- 📄 **PDF/DOCX Generation** for all templates
- 👩‍⚖️ **Lawyer Recommendations & Connect Panel**
- 🔒 **Secure Document Storage**
- 🧠 **OpenAI/GPT Integration** for contextual legal intelligence
- 🌐 **Responsive Frontend** (React)
- 🎯 **Robust error handling & validations**

## 🧑‍💻 User Flow

1. **Visit Homepage**  
   View introduction and available options

2. **Select Role/Use Case**  
   Choose from HR, SEO, Accountant, or other categories

3. **Fill Document Generation Form**  
   Dynamic form collects role-based, contextual data

4. **(Optional) Chatbot Support**  
   Ask legal questions, get real-time suggestions

5. **AI Processes Request**  
   Form data sent to backend; GPT + templates generate legal content

6. **Document Output**  
   Download, preview, or send as PDF/DOCX; option to connect with a lawyer

7. **Lawyer Connect (Optional)**  
   Choose a lawyer from the “Corner of Experts” for secure communications

## ⚙️ Getting Started

### 1. Clone the Repository

```bash
git clone https://github.com/Sudesh-chandra/Lawgorithm--AI-Legal-Assitant.git
cd Lawgorithm--AI-Legal-Assitant
```

### 2. Backend Setup

```bash
cd server
npm install
npm start
```

Create a `.env` file in `/server` with:

```
OPENAI_API_KEY=your_openai_key
MONGO_URI=your_mongodb_uri
PORT=your_port
```

### 3. Frontend Setup

```bash
cd ../client
npm install
npm start
```

> **Note:** Update the API base URL in `api.js` if needed.
